# **App Name**: GenSite

## Core Features:

- AI-Powered Content Generation: Automatically generate all website text using AI. This feature allows the site to produce inspiring and knowledgeable content without manual input. The tool takes into consideration your selected site name to suggest text, and it can iterate on these ideas when prompted.
- AI-Driven Layout Design: The AI designs the website layout automatically, ensuring a premium, minimal, and aesthetic look. The AI chooses component arrangements with mobile-friendliness as a priority.
- Automated Image Selection: AI selects relevant images and icons for the website based on the content and design, enhancing the visual appeal and user experience.
- Responsive Design: Website adapts seamlessly to different screen sizes, providing an optimal viewing experience on all devices.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) to inspire creativity and knowledge.
- Background color: Very light violet (#F3E5F5), a desaturated tint of the primary, for an airy, spacious, premium background.
- Accent color: Darker indigo (#3F51B5) to complement the violet and give contrast on buttons and selected elements.
- Body and headline font: 'Alegreya', a serif font giving an elegant, intellectual, contemporary feel, well-suited to body or headlines.
- Minimalist design with plenty of whitespace to enhance readability and create a premium feel.
- Simple, elegant icons that complement the overall aesthetic and enhance user navigation.
- Subtle transitions and animations to create a smooth and engaging user experience.
